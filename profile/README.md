# Forky-s

Curated forks of Swift libraries used across apps and tools

This organization keeps dependency forks in one place so projects can rely on stable mirrors, targeted fixes, and preserved package history when upstream availability or release timing becomes a problem

## Focus

- Swift and SwiftUI packages
- macOS and iOS app dependencies
- Utility libraries used across multiple projects
- Forks kept close to upstream unless a project needs a focused change

## Notes

Most repositories here are forks of upstream open source projects. Check each repository for its original source, license, and upstream project history
